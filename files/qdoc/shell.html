Detailed Description

Class for create a javascript shell.

The Shell class allow to create javascript shell in console or with GUI.

QStringList Shell::completeScriptExpression ( QString expression, int & completitionStartAt, QString & commonName )

Find completitions of a javascript expression. Find and return all posible completitions of a javascript expression. As output arguments completitionStartAt contains the positions at the expression when the completion posibilities start and commonName contains the common text between all posibilities of completition.
