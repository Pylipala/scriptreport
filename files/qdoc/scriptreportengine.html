Detailed Description

Class for render the resultant HTML of a Script Report.

The ScriptReportEngine class is the engine of Script Report, this class render and print the resultant HTML of a report. This class exists for add more backends (not supported at this momment) that QTextDocument.

Warning: Use ScriptReport for run a Script Report.
