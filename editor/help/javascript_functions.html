Script Report Engine

sr.engine

Class for interact with the script engine and to do basic operations.

List of basics commands:

Method Description
error([message, ...])print a message in the standard error.
exit() exit the shell with return code 0 (zero).
exit(code) exit the shell with return number code pased as the argument code.
getFromEnvironment(name)search for a variable identified by name and returns its value or null if is not found.
help() display this help with the basic usage.
importExtension([name, ...])load the extension(s) with name name.
load([filename, ...])load and excute the javascrip file(s) with name name inside the current shell.
print([message, ...])print a mensagge(s) in the standard output.
quit() exit the shell with return code 0 (zero).
read([message, ...]) read a line from the standard input. If a message this will be printed an then will read the input.
readFile(name) read all content of a file.
runCommand(commandName, [arg, ...] [options])execute the specified command with the given argument and options as a separate process and return the exit status of the process. See runCommand usage for more detail.
runCommand(options) execute the specified command in the options object as a separate process and return the exit status of the process. See runCommand usage for more detail.

List of basic properties (exposed as a readonly variables):

Method Description
argumentslist of all argumets given at the command line when the shell was invoked.
availableExtensionslist of all avaiable QtScript' extensions.
importedExtensionslist of all imported QtScript' extensions.
qtVersionversion of the Qt library in use.
srVersionversion of the shell, value 0.1.

runCommand() usage:

runCommand(command)
runCommand(command, arg1, ..., argN)
runCommand(command, arg1, ..., argN, options)
runCommand(options)
All except the last arguments to runCommand are converted to strings and denote command name and its arguments. If the last argument is a JavaScript object, it is an option object. Otherwise it is converted to string denoting the last argument and options objects assumed to be empty.
The following properties of the option object are processed:
Method Description
argsprovides an array of additional command arguments.
envA JavaScript object with the properties to add to the process environment. All its enumeratable properties define the corresponding environment variable names.
inputthe process input. A string sent to the process as its input. If not specified, no input is provided to the process.
outputthe process output. If it is specified, the process output is read, converted to a string, appended to the output property value converted to string and put as the new value of the output property.
errthe process error output. If it is specified, the process error output is read, converted to a string, appended to the err property value converted to string and put as the new value of the err property.
commandcommand name. Only used if the command name is not passed as argument of the runCommand method.
resultexit statud of the process. If it is specified, the exit statud of the process will be stetted in this property.

Script Report

sr.report

Class for configure the report. The title property set the report title, used for set the print job name; the page property return the current page number, the pageCount property return the number the pages of the document. The isEditing property tells if the editing mode is enable (e.g. inside the Script Report Editor), the isFinal tells if the editing mode is not enabled; the isDebbuging tells if the debbuging mode is enabled (e.g. running the debbuger inside the Script Report Editor).

Sql plugin

QScriptExtensionPlugin that provide QSql wrapper classes. For use this plugin import the srsql extension.

By default, all wrapper class throw an error when a sql error has happened, for disable this behavior, set autoThrow property to false.

Important: All database connections opened will be closed when the script engine is destroyed; in consecuence, all connections created in the script will be removed.

Internationalitation plugin

QScriptExtensionPlugin that provide internationalization wrapper classes, like QLocale, QTranslator, or the ability to install translators and the translator functions. For use this plugin import the sri18n extension.

sr.i18n

Wrapper class around QLocale's static members, and also, this class allow to install translator and translator functions, and openTranslator method (like to QTranslator load method).

Note: the installApplicationTranslatorFunctions function works with the QScriptEngine translators functions, the installTranslatorFunctions function work with its own implementation.

Note: the installApplicationTranslator and installPermanentApplicationTranslator methods install translator using QCoreApplication, installTranslatorFunctions methods install translators internally (not use QCoreApplication and don't generate a LanguageChange event), but these translators only are avaiable installing the translators functions using installTranslatorFunctions method.

Important: All installed translators will be removed when the script engine is destroyed, except all translators installed using installPermanentApplicationTranslator methods.
